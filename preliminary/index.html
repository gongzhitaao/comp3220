3.1 Example – Readability/Writability

Java reads a file. 

BufferedReader br = new BufferedReader(
                      new InputStreamReader(
                        new FileInputStream(
                          new File("file.txt"))));
String line = null;
while ((line = br.readLine()) != null)
    ;                           // read line by line

Python reads a file. 

with open("file.txt") as fin:
    for line in fin:
        pass                    # read line by line

3.2 Example – Reliability

{
  // I'm a comment

}
